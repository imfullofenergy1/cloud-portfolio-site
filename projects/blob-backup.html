☁️ Azure Blob Backup Script

← Back to Portfolio

This Python script automates file uploads to Azure Blob Storage using the Azure SDK. Great for backups and automation tasks.

🛠️ Features

  • Creates a container if it doesn't exist
  • Uploads a local file to Azure Blob
  • Uses the official Azure SDK (BlobServiceClient)
  • Command-line usage with customizable file path
