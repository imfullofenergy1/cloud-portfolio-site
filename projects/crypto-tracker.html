💹 Crypto Portfolio Tracker

← Back to Portfolio

This project is a Flask web app that fetches real-time ETH and token balances using the Etherscan API and displays them in a clean dashboard.

🔧 Features

  • Connects to MetaMask or Trust Wallet
  • Shows real-time ETH + token balances
  • Pulls data from Etherscan
  • Built with Flask, HTML/CSS, and Python
  • Secure `.env` file for API key privacy
